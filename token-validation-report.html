How to Fix

Replace hardcoded color values with appropriate tokens from the design system:

  • For SCSS files, use token functions: color('primary')
  • For React components, use CSS variables: var(--color-primary)
  • For RGBA values with opacity: rgba(var(--color-primary-rgb), 0.5)

See the Color Token Documentation for a complete reference.
